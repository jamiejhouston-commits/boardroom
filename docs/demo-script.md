# Boardroom — 60-second demo shot list

Goal: one continuous-feeling phone recording that makes a stranger say
"wait, it does WHAT on its own?" Record vertical (it'll live on X/TikTok and
embed in the README). No talking-head needed — screen + captions carry it.

**Recording setup (no filming skills needed):** iPhone → Mac via cable →
QuickTime → File → New Movie Recording → arrow next to ⏺ → select iPhone as
camera AND microphone (captures the agents' voices). Record the whole run in
one take; trim + captions in CapCut (free); export 1080×1920. Raw screen
capture is the right look — do NOT over-produce it.

**Prep (before recording):**
- Relay running (`./Scripts/setup.sh`), company HALTED, one initiative already
  sitting at **gate1** (so the greenlight moment is ready) and a second one at
  **gate2** if possible (so Demo Day is ready). Phone volume UP.
- Set heartbeat interval to 5 min beforehand so stages move during/right after
  recording if needed.
