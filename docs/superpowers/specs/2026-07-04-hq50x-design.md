# HQ 50x — Living Headquarters (design + plan)

Date: 2026-07-04 · Status: APPROVED by owner ("GO") after conversational design review.

## Goal
Turn the SceneKit Headquarters (Views/Headquarters/) from a cinematic diorama into a
living, connected command floor: free-roam walking, in-room agent conversations with
speech bubbles + voice, a fully staffed floor, and live company data physically in
the room. Orbit/Overview/Inspect stay.

## Constraints (non-negotiable)
- Muted premium palette only (HermesTheme/HQSceneBuilder values). No neon.
- iPhone GPU budget: no new SCNLights, agent cap 8, SK textures ≤ 2048px,
  bubbles are SwiftUI overlay (zero scene cost), existing bloom/AA untouched.
- Only the owner builds/runs. Claude verifies via xcodebuild compile-check
  (no simulator launch) + pure-logic unit tests.
- Existing camera modes and HQLayout invariants (CEO → executive, first two
  managers → pods) preserved.

## Architecture

### 1. Roam (walk mode)
- `HQRoam.swift` — pure `HQRoamMath.step(state, stick, look, dt)` (simd, unit-tested):
  yaw/pitch integration, wall bounds clamp, executive-platform block, fixed eye
  height 1.6. `HQRoamControl` (@unchecked Sendable, NSLock) bridges SwiftUI
  joystick + UIKit pan → SceneKit render loop.
- `HQCameraMode.roam` new case. Coordinator becomes `SCNSceneRendererDelegate`;
  per-frame `control.step(now:)` drives `cameraNode` directly (render thread).
- HUD: virtual joystick (bottom-left, roam only) = move; one-finger pan on the
  scene = look. Tap still selects.

### 2. In-room conversation (bubbles + voice)
- Tap agent → NO AgentChatView sheet. `HQConversationModel` (@MainActor) streams
  via the proven independent pattern (AgentCallView/MeetingHub): same
  `agent.chatRouting` session/profile (same brain as chat + company),
  `CompanyContext.brief` prepended, persona intro on first turn, `fast:false`
  typed / `fast:true` voice.
- `HQConversationOverlay` (SwiftUI, in HUD layer): comic bubble (fixed anchor —
  inspect framing centers the agent; zero per-frame projection cost), streaming
  text, input bar, hold-to-talk mic (VoiceNoteRecorder STT → stream →
  AgentVoice.speak Piper), "Full chat" escape hatch to AgentChatView.
- `HQAgentNode.enterConversation(facing:)`: stop stroll, turn to camera
  (yaw = atan2(-dx,-dz) — model forward is -Z), Wave-if-clip-exists, Idle.

### 3. Staffed floor (3 → 8)
- New archetypes + anchors: commandEast/commandWest (flanking the dais),
  researchLab2/engineeringDen2 (second pod desks), lounge. Fill order after
  CEO+managers; cap 8. HQLayoutTests updated (cap 8 + new invariants).

### 4. Live boards (SpriteKit textures on SceneKit planes)
- `HQLiveBoards.swift`: War Board (east wall — initiatives, stageLabel,
  progress bars), Kanban (west wall — todo/doing/done), news Ticker (north wall
  marquee — CompanyEvents), Decision Desk (gold pedestal near exec steps —
  pendingGates badge, subtle pulse only when >0). Named nodes `hq.tap.*` →
  tap routes to sheets: initiatives list (rows → InitiativeRoomView), tasks
  sheet, gate sheet (Approve/Revise/Kill via relay.companyGate).
- Updates only when a company-state signature changes (15s cadence).

### 5. Alive + connected
- HeadquartersView: 15s refresh loop (today: one-shot).
- Meeting choreography: liveMeeting attendees walk to dais ring, face center;
  walk home when it ends. HUD banner: 🔴 topic + latest turn.
- Ship celebration: newly `shipped` initiative → agents dance (clip-guarded) +
  brief muted confetti burst.
- Time-of-day: ambient/sun warmth+intensity tuned by local hour (subtle).
- Office life: existing stroll kept; conversation/meeting take priority.
